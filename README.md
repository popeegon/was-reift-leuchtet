*What ripens, gleams* is a browser-based generative artwork that transforms chat conversations into a moving topology of recurrence, proximity, density, memory and connection.
The entered text is analysed locally in the browser. No language model, external AI evaluation, API or upload is used. The work does not summarise or interpret the conversation semantically; it visualises formal traces: repetition, distance, return, tension and sediment.
The result is neither a dashboard nor a word cloud, but a rule-based image event: a possible body of the text.

*Was reift, leuchtet* ist ein browserbasiertes generatives Werk, das Chatverläufe in eine bewegte Topologie aus Wiederkehr, Nähe, Dichte, Erinnerung und Verbindung überführt.
Der eingegebene Text wird lokal im Browser ausgewertet. Es gibt kein Sprachmodell, keine externe KI-Auswertung, keine API und keinen Upload. Das Werk fasst den Chat nicht zusammen und deutet ihn nicht semantisch; es visualisiert formale Spuren: Wiederholung, Abstand, Rückkehr, Spannung und Sediment.
Das Ergebnis ist weder Dashboard noch Wortwolke, sondern ein regelbasiertes Bildereignis: eine mögliche Gestalt des Textes.
